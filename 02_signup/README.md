## 安装依赖的模块

<!-- 最新版本已经把 body-parser 集成进去了 -->

```
cnpm i express cors jsonwebtoken cookie-parser morgan mongoose bcryptjs -S
```

- express

- morgan 记录访问日志

- jsonwebtoken 前后台分离时权限认证的一套方案

- bcryptjs 加密密码

- mongoose

- cors 让服务器支持跨域
